# 📚 Recapitulação Java

Este repositório reúne exercícios, exemplos e anotações utilizados para revisar os principais conceitos da linguagem Java.

## Conteúdos

- Variáveis
- Operadores
- Arrays
- POO (Classes, Objetos, Herança, Polimorfismo, Encapsulamento e Abstração)
- Construtores
- Getters e Setters
- Collections
- HashSet
- Generics
- Records

## Objetivo

Consolidar os conhecimentos em Java por meio de exercícios práticos e revisões dos conceitos fundamentais.
